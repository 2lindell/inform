A simple build and package manager for the Inform tools. This is version 1.

  • Manual

    • Using Inbuild - A brief user guide.

  • Chapter 1: Delegation

    • Main - The top level, which decides what is to be done and then carries this plan out.

Together with the following imported modules:

  • foundation - A library of common code used by all of the Inform tools.

  • words - A library of code for reading natural language into a stream of words.

  • syntax - A library of code to build a parse tree and break text into sentences.

  • html - HTML and Javascript generation tools.

  • arch - Definitions of Inter and final VM architectures.

  • inbuild - The functional part of the build manager.
