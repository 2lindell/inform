  • codegen

An embryonic code generator for I6 code.

  • Chapter 1: Setting Up

    Loading this module.

    • Codegen Module - Setting up the use of this module.

    • Stage Control - To control the stages through which code generation proceeds.

  • Chapter 2: Linking

    • Cache CM - To cache inter definitions of rules from a given compilation module.

    • Import - To import inter from a secondary file.

    • Linker - To link inter from I7 with template code.

    • I6 Template Reader - Inform 6 meta-language is the language used by template files (with extension |.i6t|). It is not itself I6 code, but a list of instructions for making I6 code: most of the content is to be copied over verbatim, but certain escape sequences cause Inform to insert more elaborate material, or to do something active.

    • Parse Linked Matter - To generate the initial state of storage for variables.

    • Resolving Conditional Compilation - To generate the initial state of storage for variables.

    • Assimilate Linked Matter - To generate the initial state of storage for variables.

    • Uniqueness - To make sure certain symbol names translate into globally unique target symbols.

    • Reconcile Verbs - To reconcile clashes between assimilated and originally generated verbs.

    • Eliminate Redundant Matter - To reconcile clashes between assimilated and originally generated verbs.

  • Chapter 3: Generating Inform 6

    • Code Generation - To generate I6 code from intermediate code.

    • Instances and Properties - To generate the initial state of storage for instances and their properties, and all associated metadata.

    • Variables - To generate the initial state of storage for variables.
