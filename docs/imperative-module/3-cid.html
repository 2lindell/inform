§1.3.1. In Inter, all functions must return a value: in Inform 7, some phrases do not. If we are compiling a function to perform such a phrase, we have it return 0. This value will almost certainly be thrown away, but it seems clearest to make it 0 in all cases.

Otherwise, if execution reaches the end of our function, we return the default value for its return kind: for example, the empty text for K_text.
