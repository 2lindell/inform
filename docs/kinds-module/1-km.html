Setting up the use of this module.

§1. This section simoly sets up the module in ways expected by foundation, and contains no code of interest. The following constant exists only in tools which use this module: 

    define KINDS_MODULE TRUE

§2. To begin with, this module needs to allocate memory: 

    enum dimensional_rule_array_MT
    enum kind_array_MT
    enum kind_variable_declaration_MT
    enum kind_constructor_MT
    enum kind_template_definition_MT
    enum kind_macro_definition_MT
    enum kind_template_obligation_MT
    enum kind_constructor_comparison_schema_array_MT
    enum kind_constructor_casting_rule_array_MT
    enum kind_constructor_instance_array_MT
    enum unit_sequence_array_MT

    ALLOCATE_IN_ARRAYS(dimensional_rule, 100)
    ALLOCATE_IN_ARRAYS(kind, 1000)
    ALLOCATE_INDIVIDUALLY(kind_variable_declaration)
    ALLOCATE_INDIVIDUALLY(kind_constructor)
    ALLOCATE_INDIVIDUALLY(kind_macro_definition)
    ALLOCATE_INDIVIDUALLY(kind_template_definition)
    ALLOCATE_INDIVIDUALLY(kind_template_obligation)
    ALLOCATE_IN_ARRAYS(kind_constructor_casting_rule, 100)
    ALLOCATE_IN_ARRAYS(kind_constructor_comparison_schema, 100)
    ALLOCATE_IN_ARRAYS(kind_constructor_instance, 100)
    ALLOCATE_IN_ARRAYS(unit_sequence, 50)
