  • html

Support for HTML problem, index and extension documentation pages. This is version 1.

  • Preliminaries

    • What This Module Does - An overview of the html module's role and abilities.

    • How To Include This Module - What to do to make use of the html module in a new command-line tool.

  • Chapter 1: Starting Up

    • HTML Module - Setting up the use of this module.

  • Chapter 2: Intranet

    Non-standard HTML utility functions for generating the Inform app intranets.

    • Source Links - To provide HTML links to an Inform source text, using the non-standard "source" protocol.

    • Paste Buttons - HTML for a button which, when clicked, pastes Inform source text into the Source panel of the application, or opens a file or folder on the host computer.

    • Documentation Renderer - To render a passage of extension documentation as HTML.

    • Installed Files - Filenames for a few unmanaged files included in a standard Inform installation.

    • Documentation References - To enable index or results pages to link into documentation.

    • Localisation - Utility functions for standing text which may vary by language.
