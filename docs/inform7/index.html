  • inform7

The core compiler in a natural-language design system for interactive fiction. This is version 10.1.0.

  • Manual

    • Command-Line Usage - A brief note on using Inform 7 at the command line.

    • Reference Card - The current help information as it would be printed at the command line.

  • Chapter 1: Delegation

    • Main - A shell for the modules which actually form the compiler.

Together with the following imported modules:

  • foundation - A library of common code used by all of the Inform tools.

  • inbuild/words - A library of code for reading natural language into a stream of words.

  • inbuild/syntax - A library of code to build a parse tree and break text into sentences.

  • inflections - A library for adapting words or short phrases for linguistic purposes.

  • problems - The issuing mechanism for problem messages used by Inform.

  • linguistics - A library of code to store grammatical categories of vocabulary.

  • kinds - The type system used by Inform.

  • inbuild/html - HTML and Javascript generation tools.

  • inbuild/arch - Definitions of Inter and final VM architectures.

  • inter/inter - Code for dealing with intermediate-level Inform.

  • inbuild/inbuild - The functional part of the build manager.

  • core - The core of the Inform compiler, as a module.

  • inter/building - An intermediate layer for building code in Inter.

  • inter/codegen - The code generation pipeline.

  • if - The interactive-fiction specific layer of Inform, as a module.

  • multimedia - A multimedia extension module for Inform.

  • index - Indexing tools needed by Inform, as a module.
