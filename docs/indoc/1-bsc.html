§1. Who we are. This is a command-line tool built on top of the Foundation module. The first definition we need to make is: 

    define INTOOL_NAME "indoc"

    void Basics::start(void) {
        Foundation::start();
        <Declare the debugging log aspects 1.2>;
    }

    void Basics::end(void) {
        Foundation::end();
    }

The function Basics::start is used in 1/mn (§1).

The function Basics::end is used in 1/mn (§1).

§1.1. Just two logging aspects to add to the usual Foundation stock: 

    enum SYMBOLS_DA
    enum INSTRUCTIONS_DA

§1.2. <Declare the debugging log aspects 1.2> = 

        Log::declare_aspect(SYMBOLS_DA, L"symbols", FALSE, FALSE);
        Log::declare_aspect(INSTRUCTIONS_DA, L"instructions", FALSE, FALSE);

This code is used in §1.
