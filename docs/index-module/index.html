  • index

Indexing tools needed by Inform, as a module. This is version 1.

  • Chapter 1: Starting Up

    • Index Module - Setting up the use of this module.

  • Chapter 2: Indexing

    Utility routines for generating index pages.

    • Index File Services - To provide routines to help build the various HTML index files, none of which are actually created in this section.

    • Documentation References - To enable index or results pages to link into documentation.

    • Lexicon Index - To construct the Lexicon portion of the Phrasebook page of the Index, which gives brief definitions and references for nouns, adjectives and verbs used in source text for the current project.

    • Index Physical World - This section masterminds the creation of the World and Kinds index pages, though it delegates much of the work elsewhere. Though it does belong to core Inform, these indexes will look pretty sparse if the spatial Plugins aren't plugged in.

    • Kinds Index - To produce most of the Kinds page in the Index for a project: the chart at the top, and the detailed entries below.

    • Index Extensions - To keep details of the extensions currently loaded, their authors, titles, versions and rubrics, and to index and credit them suitably.

    • Index Headings - To produce the index contents listing and the XML headings file.
